## :bulb: About 

Desafio 01/02 do bootcamp **Ignite 2021** da trilha de **React Native**.

<br />

## :gear: Iniciando

```Bash
# 1. Instalar dependências
yarn install

# 2. Executar metro-bundler
yarn start

# 3. Executar app
yarn <android - ios>
```
